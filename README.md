# AI Posture Detection System

This project detects human posture using computer vision and classifies it as good or bad posture in real time.

## Features
- Real-time posture detection
- Uses MediaPipe Pose
- No dataset required

## Tech Stack
- Python
- OpenCV
- MediaPipe

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python src/main.py
